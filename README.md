# Repush

Repush (a play on *retouche*) is a tool that removes your name out of all your commits, like [Redact](https://redact.dev/) would do.

## Installation

1. Install Python 3.13+
2. Clone the repository to your local device
3. `cd repush && pip install -r requirements.txt`
4. Set `GH_TOKEN` environment variable to your GitHub token. [(get one here)](https://github.com/settings/tokens)
5. Run the script `python3 app.py` and follow the instructions on screen.

## Warning

GitHub caches a lot of stuff and it takes time (according to the internet, 24 hours) to update. Please take this into a consideration. I bear no responsibility for your action.

Also, please keep in mind there are **other ways** to get older version of the repository and review further. It requires knowledge, technical skill and time, but it is possible.
